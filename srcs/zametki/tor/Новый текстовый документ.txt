Быстрый рецепт для приготовления свежего сайта в сети Tor c использованием веб-сервера Nginx.

<!--more-->

Создать сайт в даркнете не так сложно, как может показаться на первый взгляд. В сети Tor существует огромное множество сайтов, включая личные блоги и зеркала топовых сайтов, таких как Facebook или Twitter. Сегодня создание своего сайта в торе не является чем-то девиантным, напротив если вы живете в странах с сильной цензурой, это может быть очень полезно.
Главными преимуществами луковой маршрутизации является высокая степень анонимности сайта и пользователя а также невозможность блокировки надзорными органами государств.

Ниже приведен чеклист, для быстрого запуска вашего сайта в сети Тор:

# Чеклист

0. Понадобится VPS с установленной серверной Ubuntu или Debian 10 и новее (?)
1. Устанавливаем Tor
   ```Bash
   sudo apt install tor
   ```
2. Настраиваем конфиг .torrc
   ```Bash
   sudo vim /etc/tor/torrc
   ```
   или
   ```Bash
   sudo nano /etc/tor/torrc
   ```
   Раскомментируем строки:
   ```Bash
   HiddenServiceDir /var/lib/tor/hidden_service/
   HiddenServicePort 1337 127.0.0.1:1337
   ```
   или
   ```Bash
   HiddenServicePort 1337 unix:/var/run/nginx.sock
   ```
3. Запускаем Tor
   ```Bash
   cmd sudo tor
   ```
4. Узнаем сгенерированное имя .onion домена
   ```Bash
   cmd sudo cat /var/lib/tor/hidden_service/hostname
   ```
5. Настраиваем Nginx для работы с даркнетом
   ```Nginx
server {
     listen 127.0.0.1:1337;
     server_name xyew6pdq6qv2i4sx.onion;
     allow 127.0.0.1;
     deny all;
     server_tokens off;
     location / {
       ...
     }
   }
```
